# dashboard-ventas-powerbi

**🧠 Caso de estudio: **Dashboard de Ventas y Cumplimiento de Metas****

🔹 Descripción del proyecto

Desarrollé un dashboard interactivo en Power BI para analizar el desempeño de ventas en comparación con metas definidas, incorporando análisis por región, producto y evolución temporal.

🔹** Desafío principal**

El principal desafío fue construir un modelo de datos correcto que permitiera relacionar ventas con metas sin generar errores en los cálculos, especialmente en el indicador de cumplimiento, el cual inicialmente presentaba inconsistencias.

🔹 **Solución propuesta**

Se diseñó un modelo de datos optimizado, ajustando las relaciones entre tablas y corrigiendo medidas DAX para evitar duplicaciones y errores de contexto. Además, se implementaron visualizaciones interactivas con segmentadores para facilitar el análisis dinámico.

🔹 **Herramientas utilizadas**

Power BI
Power Query
DAX
Excel / CSV

🔹 **Principales aprendizajes**

Durante el desarrollo comprendí la importancia del modelado de datos en Power BI, especialmente cómo las relaciones afectan directamente los resultados. También aprendí a utilizar DAX para construir métricas más precisas y a diseñar dashboards enfocados en la interpretación de datos.

🔹 **Métricas de impacto**

Construcción de dashboard interactivo completo
Integración de múltiples fuentes de datos
Automatización de cálculos (KPIs)
Mejora en la visualización y análisis de información

🔹 **Habilidades aplicadas**

Modelado de datos
DAX (medidas y cálculos)
Visualización de datos
Análisis de negocio
Resolución de errores en contexto de datos

🔹 Justificación del proyecto
Elegí este proyecto porque representa de forma clara mi capacidad para trabajar con datos, construir soluciones analíticas y transformar información en insights útiles para la toma de decisiones.
